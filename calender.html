Redesign NEU web calendar Application (Wireframe)

  • Type: Individual project (for Usibility of Human Interaction project)
  • My Role: Ideation, wireframes
  • Method: desktop wireframes
  • Tool: | Balsamiq |
  • Timeline: OCT 2016 (2 Weeks)

Process Overview

A common complaint of students is that while there are many activities going on at Northeastern each day, it’s difficult to find timely information about campus events. There is currently an online campus event calendar that can be found at http://calendar.northeastern.edu. I was tasked with designing an improved version of this calendar application to better serve student’s needs.

The Problem

Harmonix Music Systems is a video game development company based in Cambridge, Massachusetts in the United States. But their website interface leaves a lot to be desired.I improved the navigation and added some nifty new features along the way.
